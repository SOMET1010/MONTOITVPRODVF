Analyser les fonctionnalités des principales plateformes immobilières françaises (Leboncoin, SeLoger, PAP, MeilleursAgents, Century21) pour identifier les bonnes pratiques et innovations à intégrer dans MONTOIT.

- [ ] Fonctionnalités innovantes

1. [ ] Recherche web pour obtenir les URLs officielles
2. [ ] Analyse détaillée de chaque plateforme via extraction de contenu

4. [ ] Analyse comparative des approches
5. [ ] Synthèse des bonnes pratiques pour MONTOIT
6. [ ] Rédaction du rapport final
